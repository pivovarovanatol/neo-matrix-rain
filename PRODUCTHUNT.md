## Long description (for PH body)

---

**Hey hunters! 👋**

I built Neo Matrix Rain — a lightweight, native macOS screensaver that fills your screen with the iconic falling character rain you know from the films.

**Why another Matrix screensaver?**

Most alternatives are:
- Electron wrappers (heavy, slow)
- Web-based (drain battery, no native feel)
- Old and unmaintained (broken on Apple Silicon)

This one is different:
- ✅ Built in Swift + CoreGraphics — fully native
- ✅ Runs on Apple Silicon and Intel
- ✅ No dependencies, no frameworks, no web views
- ✅ Tiny — 250KB install
- ✅ macOS 12+ (Monterey, Ventura, Sonoma, Sequoia)
- ✅ Free & open source (MIT)

**One-click install** — download the .zip, double-click, done. No Xcode required.

**What you get:**
- Falling katakana characters (the authentic Matrix look)
- Cyrillic characters (unique twist)
- Digits mixed in
- Green gradient trail with realistic glow
- Smooth, CPU-friendly animation

**What's coming:**
- Settings UI — adjust speed, density, colors without touching code
- Color themes — blue, red, white, custom RGB
- Custom character sets

If you enjoy it, a coffee on Ko-fi keeps development going ☕
